1. Install Git :"http://git-scm.com/download/win".
2. Open git from programs and enter the following commands:
  a) git config --global user.name "Your Name"
  b) git cofig --global user.email "email@gmail.com"

3. Then you will see that a new folder has been created.

Now run the following commands each time you want to work in this project:

1. git pull origin master
2. git status
3. git add .
